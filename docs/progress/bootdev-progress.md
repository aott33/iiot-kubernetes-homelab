## Why Boot.dev?

I'm using the Boot.dev curriculum to build a strong foundation in backend development and cloud-native technologies, which directly supports this IIoT homelab project:

- **Go Programming** - Core language for services in this project
- **Docker & Kubernetes** - Container orchestration used throughout
- **HTTP & APIs** - RESTful APIs for platform integration
- **SQL & Databases** - Time-series and relational data storage
- **Security** - Critical for OT/IT network segmentation
- **CI/CD** - Automated testing and deployment pipelines
